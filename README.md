## Задание 1.

В задании 1 необходимо описать класс, который реализует функционал неупорядоченного списка
(обратиться к элементу по индексу нельзя), который может содержать только уникальные элементы
(в случае добавления повторяющихся значений ошибки нет, такой элемент просто не добавляется).
Полем класса является массив. В классе должны присутствовать методы добавления и удаления
элементов, а также методы: <br>
• Union() — объединяет две коллекции;<br>
• Except() — позволяет удалить из первой коллекции элементы второй;<br>
• Intersect() — получает коллекцию, состоящую из элементов, общих для двух коллекций;<br>
• Contains() — проверяет на наличие элемента, возвращает bool.

Есть перечень торгово-развлекательных центров (ТРЦ) города. Студенты группы любят посещать
ТРЦ. Известно для каждого студента, в каких ТРЦ он побывал. Определить:<br>
• в какие ТРЦ из перечня ходили все студенты группы;<br>
• в какие ТРЦ из перечня ходили некоторые студенты группы;<br>
• в какие ТРЦ из перечня не ходил ни один из студентов группы?


## Алгоритм решения:

Создаем множество торговых  центров, массив students, где каждый элемент — это личный список ТРЦ конкретного студента. 

Чтобы найти общие элементы для всех студентов, используется операция Intersect. Переменная all сначала получает копию списка первого студента. Делаем цикл for начиная со второго студента. Внутри метода Intersect: создается пустое временное множество result, Запускается цикл for по элементам текущего списка, проверяется nextStudent.Contains(), который ищет пересечение. На следующей итерации этот короткий список будет сравниваться со следующим студентом.

Чтобы найти список тц, которые посетил хотя бы один студент, используем Union. Создается пустое множество some. В цикле в него добавляются (методом Add) все тц каждого студента.

Чтобы найти тц, в которых ни один студент не был, используем операцию Except. Берется полный список тц и из него вычитается множество some. Внутри метода Except: создается пустое множество result, запускается цикл по всем тц, для каждого тц вызывается some.Contains() == false, и если условие выполнено, то добавляем элементы в результат.
